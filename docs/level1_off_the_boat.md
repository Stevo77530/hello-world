# Level 1 Design Stub — Off the Boat

## Level Intent
Teach movement/combat basics while establishing the anti-home world and ending in a boss gate denial event.

## Target Flow (First ~3 Minutes)
1. Intro cinematic
2. Cabin / ready room tutorial
3. Interior corridor
4. Outer hatch / deck access
5. Deck traversal
6. Gangway
7. First enemy reveal on dock
8. Dock/yard push toward exit gate
9. Boss arena at departure gate

## Room-by-Room Breakdown

### 1) Black Screen Intro Space (non-playable trigger)
- Audio-first narrative setup
- Locked dialogue sequence executes
- Ends on objective card: **GET OFF THE BOAT**

### 2) Cabin / Ready Room (safe tutorial pocket)
- Player gains control
- UI prompt placeholders for movement, look, interact, fire
- Environmental storytelling: personal locker, union notices, corrupted relief terminal

### 3) Interior Corridor
- Narrow pressure hallway with flicker lights and intercom artifacts
- First micro-threat hints (distant footsteps, radio static, signage corruption)

### 4) Outer Hatch / Deck Access
- Interaction beat: manual hatch open
- Audio spike when transitioning from enclosed interior to exposed deck

### 5) Deck Traversal
- Mid-range sightlines
- Introduce traversal pressure and hazard telegraphing (sparks, swinging chains)

### 6) Gangway Transition
- Funnel encounter space
- Teases dock activity below and route to gate

### 7) First Enemy Reveal (Dock Corrupted)
- Staged reveal on dock floor with clear silhouette read
- One simple skirmish to teach strafing and range control

### 8) Dock/Yard Push
- Short sequence of cover objects and flanking lanes
- Signage/intercom inject signal phrases:
  - NO CREW CHANGE
  - RETURN TO VESSEL
  - RELIEF UNAVAILABLE

### 9) Departure Gate Boss Arena
- Gate opens slightly (fake hope)
- Intercom denial line triggers
- Gate slams shut
- Boss enters and fight begins

## Dialogue Trigger Beats (Locked)
### Opening cinematic
- Woman on phone: “Something’s wrong… hurry… come home…”
- Static whisper: “You’re never making crew change, Captain.”
- Capt. Stevo: “What the fuck?”
- Shatter SFX
- Shotgun rack SFX

### Boss intro at gate
- Intercom: “Your crew change request has been reviewed.”
- Beat
- Intercom: “We’re short-handed, so fuck you.”

## Boss Arena Summary
- Arena shape: rectangular dock gate yard with lateral cover and center pressure lane
- Initial objective fake-out (open gate) converts to forced combat lock
- Boss uses close-mid control attacks and denial burst area pressure
- Exit unlocks only after boss death line completes

## What the Player Should Learn in Level 1
1. Capt. Stevo’s world is systemically hostile, not randomly haunted.
2. Core movement/combat rhythm (move, peek, punish, reposition).
3. Signal phrases are actionable narrative warnings, not flavor text.
4. Every level culminates in a boss-level authority confrontation.
5. “Crew change” is both objective and thematic conflict engine.
